## Run Locally

Clone the project

```bash
  git clone repo_link
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`MONGODB_SECRET`

`NODE_ENV`
